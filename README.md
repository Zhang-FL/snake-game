# 贪吃蛇 Snake Game

一个使用原生 HTML/CSS/JavaScript 构建的经典贪吃蛇游戏，包含排行榜功能。

## 快速开始

直接在浏览器中打开 `index.html` 即可游玩，无需安装任何依赖。

```
snake-game/
├── index.html   # 页面结构
├── style.css    # 样式
├── game.js      # 游戏逻辑
└── README.md
```

## 玩法

- **方向键** 或 **W/A/S/D** 控制蛇的移动方向
- **空格键** 暂停 / 继续游戏
- 吃到红色食物 +10 分，蛇身变长，速度逐渐加快
- 撞墙或撞到自己则游戏结束
- 点击 **重新开始** 按钮开始新一局

## 排行榜

- 游戏结束时，如果分数能进入排行榜前 15 名，会弹出输入框记录玩家名字
- 排行榜数据存储在浏览器 `localStorage` 中，同浏览器下不同会话之间数据持久保留
- 排行榜按分数降序排列，同分按时间升序

## 技术实现

### 游戏核心

- 基于 Canvas 2D 渲染，20×20 网格，每格 20px
- 使用 `setInterval` 驱动游戏循环，初始间隔 120ms
- 每吃一个食物速度提升 1ms，最快间隔不低于 50ms
- 食物随机生成在蛇身未占据的空闲格子上

### 渲染细节

- 蛇头以圆角矩形绘制，带白色眼眶 + 黑色瞳孔的方向性眼睛
- 蛇身颜色从亮绿渐变到暗绿，呈现层次感
- 食物带有红色光晕效果
- 网格线辅助定位
- 游戏结束和暂停时有半透明遮罩 + 文字提示

### UI 布局

- 左右分栏布局：左侧排行榜面板，右侧游戏区域
- 深色主题配色（`#1a1a2e` 背景）
- 排行榜前三名分别以金/银/铜色高亮
- 当前玩家上榜记录有脉冲动画提示

## 浏览器兼容

支持所有现代浏览器（Chrome、Firefox、Safari、Edge）。
